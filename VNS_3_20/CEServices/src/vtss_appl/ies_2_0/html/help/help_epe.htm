EPE Configuration Help

Configure EPE port on this page.

To configure the EPE, Ethernet PCM encoder, select traffic can be copied, or mirrored, to the EPE port then output as a HDLC or CH7-15 encoded PCM stream.

The traffic to be copied on the EPE port is selected as follows:

  • All frames received on a given port (also known as ingress or source mirroring).
  • All frames transmitted on a given port (also known as egress or destination mirroring).

Bit Rate

Bit Rate

Bit Rate sets the bitrate to 2,000 - 20,000 kbps at 200 kbps increments.

EPE Port

EPE Port enables encoding in ISO/IEC 13239 HDLC Ethernet or CH7-15, IRIG 106-15 chapter 7 Ethernet formats. Frames from ports that have either source (rx) or destination (tx) encoding enabled are mirrored on the EPE port then ecoded as a PCM stream. Disabled disables encoding.

CH7-15: Word(s) in Minor Frame

CH7-15: Word(s) in minor frame sets the number of data words in a minor frame. Values 1-8 may be used.

Switch to mirror to

Frames from ports that have either source (rx) or destination (tx) mirroring enabled are mirrored to this switch.

EPE Port Configuration

The following table is used for Rx and Tx enabling.

Port
The logical port for the settings contained in the same row.
Mode
Select EPE mode.
Rx only Frames received on this port are encoded in the PCM stream. Frames transmitted are not encoded.
Tx only Frames transmitted on this port are encoded in the PCM stream. Frames received are not encoded.
Disabled Neither frames transmitted nor frames received are encoded.
Enabled Frames received and frames transmitted are encoded in the PCM stream.
