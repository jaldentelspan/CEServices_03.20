.TIMIN Time Input Configuration Help

TIMIN Configuration

Configure Time Input on this page.

Mode
Indicates the Time Input operation mode. Possible modes are:
IRIG-B: Enables the devices time input to receive IRIG-B time for operation.
IRIG-A: Enables the devices time input to receive IRIG-A time for operation.
IRIG-G: Enables the devices time input to receive IRIG-G time for operation.
1588: Enables the devices time input to receive 1588 time for operation.
GPS 0.0 volts: Enables the device to receive GPS time for operation without supplying a DC power source.
GPS 3.3 volts: Enables the device to receive GPS time for operation and supplying a DC power source of 3.3 volts.
GPS 5.0 volts: Enables the device to receive GPS time for operation and supplying a DC power source of 5.0 volts.
1PPS: the devices time input to receive 1PPS time for operation.
Disable: Disable the devices ability to receive time as an input
1588 enables the devices time input to receive 1588 time for operation. The GPS options, allows the device to receive time from a GPS unit and supply the option to supply DC power at 3.3 and 5.0 volts. Disable removes the functionality of the time input.
