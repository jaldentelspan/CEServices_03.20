Ethernet PCM Decoder Configuration Help

Configure Ethernet PCM Decoder port on this page.

To configure the Ethernet PCM Decoder, PCM data, HDLC or CH7-15, can be converted Ethernet traffic output on the FPGA port, port 14 on the iES-12 and port 18 on the iES-16.

Bit Rate

Bit Rate

Bit Rate sets the bitrate to 200000-50000000 bps.

EPE Port

Ethernet PCM Decoder Port allowd decoding of ISO/IEC 13239 HDLC Ethernet or CH7-15, IRIG 106-15 chapter 7 Ethernet formats.

CH7-15: Word(s) in Minor Frame

CH7-15: Word(s) in minor frame sets the number of data words in a minor frame. Values 1-8 may be used.

Invert Clock

The invert clock feature allows a device to flip the sample clock’s polarity, optimizing PCM data decoding by aligning the sampling edge with the data signal.
