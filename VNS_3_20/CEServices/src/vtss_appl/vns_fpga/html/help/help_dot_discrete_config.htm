.DISCRETE I/O Configuration

.DISCRETE Output Configuration Help

Discrete Number
Identification name for output pin.
Function
Indicates the Function to trigger output. Possible modes are:
Activity: Activates trigger when traffic is detected.
Link Up: Activates trigger when a link is established.
Time Lock: Activates trigger when the time is locked.
1588 Lock: Activates trigger when a 1588 time source is locked.
GPS Lock: Activates trigger when a GPS source is locked
Switch Error: Activates trigger when switch errors occur.
FPGA Error: Activates trigger when the FPGA detects a errors.
User Control: Enables user to manually control the output.
Disabled: Disables the discrete output.
Port
This selection is only active for the functions "Activity" and "Link Up". Activates trigger on selected port when activity or a link is detected.
Manual Trigger
This button is only active for the "User Control" function. It allows the user to manually toggle the discrete output to its alternate state.
Status
Displays status of discrete output. The status can have three different states on, off, or disabled which is respectively represented by a green, red, or grey led image.

.DISCRETE Input Configuration Help

Discrete Number
Identification name for input pin.
Enable\Disable
Indicates the Function to trigger output. Possible modes are:
Disabled: Disables the discrete input.
Enable: Enables the discrete input.
Status
Displays status of discrete input. The status can have three different states on, off, or disabled which is respectively represented by a green, red, or grey led image.
