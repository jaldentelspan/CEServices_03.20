.TIMOUT Time Output Configuration Help

.TIMOUT Configuration

Configure Time Output on this page.

Mode
Indicates the Time Output operation mode. Possible modes are:
IRIG A: Enables device to output IRIG standard A timecode.
IRIG B: Enables device to output IRIG standard B timecode.
IRIG G: Enables device to output IRIG standard G timecode.
Disable: Disables the time output mode operation.
The IRIG selections enables the output of time codes A,B,and G which were developed by the Inter-range instrumentation group. Disable removes the time output functionality.
