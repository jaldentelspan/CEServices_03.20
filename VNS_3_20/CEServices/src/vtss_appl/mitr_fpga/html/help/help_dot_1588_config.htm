.1588 Configuration

Configure .1588 on this page.

Mode
Indicates the .1588 operation mode. Possible modes are:
Master:Set.1588 mode operate as a master device.
Slave: Set.1588 mode operate as a slave device.
Disable: Disable .1588 mode operation.
When the mode is master, the device becomes a source for Precision Time Protocol (PTP) messages. In slave mode, the device becomes the destination for PTP messages. Disable deactivate .1588 functionality.

: Click to save changes.
