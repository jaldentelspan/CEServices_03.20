DSCP Remarking Configuration Help

This page allows you to configure DSCP remarking related settings on each port.
Frames can be classified into 4 different QoS classes: Low, Normal, Medium and High.
The classification can be controlled through Port QoS configuration page.
And this page is used to configure DSCP remarking.
The DSCP value of incoming frames will be changed according to its mapping queue once this packet is transmitted on the egress port.

The settings relate to the currently selected stack unit, as reflected by the page header.

DSCP Remarking Configuration

Port
The logical port for the settings contained in the same row.
DSCP Remarking Mode
If the QoS remarking mode is set to enabled, it should be with this DSCP remarking/correction function according to RFC2474 on this port.
DSCP Queue Mapping
Configure the mapping table between the queue and its DSCP value that is used for DSCP remarking if the DSCP value of incoming packets is not specified in RFC2474.
